# 지니PHP 설치 기술지원
웹 서비스 개발 플렛폼 지니PHP 설치 서비스 입니다.

> 본 기술지원은 기본 기능을 통한 서비스 운영 및 개발환경을 제공하는 기술지원 서비스 입니다. 별도의 사이트 화면 개발 및 기능구현은 커스텀 서비스를 참고하세요.

## 준비사항
솔루션을 설치하기 위해서는 독립적인 리눅스 머신이 필요로 합니다. 자체 서버 또는 클라우드와 같은 서버를 미리 준비하셔야 합니다. 

> 리눅스 운영체제 설치는 본 기술지원에 포함되어 있지 않습니다. 희망하는 경우 별도의 비용이 비용이 발생될 수 있습니다.

* 리눅스 서버
* 접속 계정

자체 제휴된 클라우드를 사용하는 경우에는 별도의 비용이 발생되지 않습니다.

## 기술지원 비용
기술비용은 `1회` 청구되며, 금액은 `$250`입니다. 
> 국내에서는 신청일 당일 환율을 적용하여 청구합니다.  


설치내역은 다음과 같습니다.
* php 설치
* 웹 서버 설치
* 데이터베이스 설치

* 라라벨 설치
* 지니PHP 설치
